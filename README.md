# Soltero Dev - Portfolio & Marketplace Platform

Professional portfolio + business marketplace platform with chat, services, and TMT currency system.

## Features
- 🎨 Professional Portfolio
- 🛍️ Service Marketplace
- 💬 Real-time Chat
- 👥 User Profiles
- 🎁 Gift Shop & TMT Currency
- 📱 PWA + Mobile App
- 🌍 Multi-language (TM/UZ/RU/EN)
- 🎯 Admin Panel
- 🔐 Gmail OAuth + JWT Auth

## Tech Stack
- Frontend: React, TailwindCSS, Framer Motion
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Hosting: Replit

## Getting Started
```bash
npm install
npm start
```

Visit: http://localhost:3000
